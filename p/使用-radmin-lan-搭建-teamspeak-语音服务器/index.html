使用 Radmin LAN 搭建 TeamSpeak 语音服务器

2025-07-04

TeamSpeak 服务器搭建指南

下载所需文件

  • TeamSpeak 客户端 ：下载地址
  • TeamSpeak 服务器 ：下载地址
  • 中文汉化包 ：下载地址

1. 安装Teamspeak并汉化

  1. 关闭 TeamSpeak 客户端
  2. 打开中文汉化包文件
  3. 重新启动 TeamSpeak 客户端
  4. 检查界面是否已变为中文

注意：

  • 安装过程中不要勾选 “Install Overwolf” 选项
  • 服务器程序只需主机安装，客户机无需安装

2. 优化音频设置

  1. 在 TeamSpeak 客户端中：点击顶部菜单的 工具 → 设置
  2. 选择 音频输入 选项，勾选 回声消除 选项

3.配置服务器

1.打开radmin lan 创建服务器,复制radminlan中主机的IP地址作为ts的服务器地址,例：26.21.69.30

2.打开team speak server，记下server admin privilege key 3.用ts连接到该服务器，输入server admin privilege key获得权限 3.点击书签 > > 添加到书签，保存服务器
